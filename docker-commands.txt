# Docker Commands - may-backend

## Build
docker build -t ozanemrahyakupoglu/may-backend:1.0.0 .

## Push (Docker Hub)
docker push ozanemrahyakupoglu/may-backend:1.0.0

## Run
docker run -d \
  --name may-backend \
  -p 8090:8090 \
  -e DB_HOST=host.docker.internal \
  -e DB_USER=MAY_APP \
  -e DB_PASSWORD=MAY_APP \
  ozanemrahyakupoglu/may-backend:1.0.0
